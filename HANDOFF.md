## üé® Design Rules & Patterns

### Scroll Choreography Patterns

**1. Reference Stagger Timing:**
```tsx
delay: index * 0.165  // Exactly 165ms between items
```

**2. Scroll Reveal Pattern:**
```tsx
variants={referenceScrollReveal(index)}
initial="initial"
whileInView="animate"
viewport={viewportConfig}  // { once: true, margin: '-20%' }
```

**3. Easing Curve:**
```tsx
ease: [0.4, 0, 0.2, 1]  // Consistent across all animations
```

**4. Duration:**
```tsx
duration: 0.6 - 0.8  // For most transitions
```

**5. Chain Reaction (Section Transitions):**
```tsx
const sectionAExit = useTransform(scrollYProgress, [0.3, 0.4], [1, 0]);
const sectionBEnter = useTransform(scrollYProgress, [0.35, 0.45], [0, 1]);
```

### Typography Rules

**Fluid Scaling (Applied):**
```css
.hero-title { font-size: clamp(3rem, 8vw, 6rem); }
.section-heading { font-size: clamp(2rem, 5vw, 4rem); }
.body-text { font-size: clamp(1rem, 2vw, 1.25rem); }
```

**Project Titles:**
```tsx
fontSize: 'clamp(3rem, 5vw, 5rem)'  // 48-80px
```

### Spacing Rules

**Vertical Spacing:**
- Section padding: `clamp(10rem, 12vw, 14rem) 0`
- Between major elements: 160px+ gaps
- Between list items: 120-160px (projects)

**Section Labels:**
```tsx
<motion.div className="text-xs tracking-widest text-gray-600 mb-20">
  (SECTION NAME)
</motion.div>
```
